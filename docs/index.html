Que es Focalboard

Organiza, planifica y colabora en un solo lugar

Focalboard es una plataforma de gestion de proyectos de codigo abierto que ayuda a definir, organizar, rastrear y gestionar el trabajo a traves de individuos y equipos.

Diseada como alternativa a herramientas propietarias como Trello, Notion y Asana, Focalboard ofrece total control sobre tus datos con la posibilidad de auto-hospedaje, soporte multilingue y una interfaz moderna e intuitiva.

Desde tareas personales hasta flujos de trabajo complejos de equipo, Focalboard se adapta a tus necesidades con multiples vistas, plantillas y propiedades personalizables.

Vistas

Multiples formas de visualizar tu trabajo

Elige la vista que mejor se adapte a tu flujo de trabajo. Cambia entre vistas en cualquier momento sin perder informacion.

K

Vista Kanban

Organiza tus tareas en columnas arrastrables. Visualiza el flujo de trabajo de tu equipo, identifica cuellos de botella y mueve tarjetas entre estados con facilidad.

T

Vista de Tabla

Visualiza todas tus tareas en un formato de hoja de calculo con columnas personalizables. Ordena, filtra y agrupa por cualquier propiedad para analisis detallados.

G

Vista de Galeria

Presenta tus tarjetas como mosaicos visuales con imagenes destacadas. Ideal para gestionar contenido visual, portafolios o cualquier proyecto con componentes graficos.

C

Vista de Calendario

Planifica y visualiza tareas en una linea temporal. Gestiona fechas limite, identifica conflictos de agenda y mantiene a tu equipo sincronizado con los plazos.

Colaboracion

Trabaja en equipo de forma eficiente

Herramientas integradas para que tu equipo colabore en tiempo real, comparta informacion y mantenga la alineacion en cada proyecto.

  • Tableros compartidos

    Comparte tableros con tu equipo y gestionen juntos las tareas del proyecto en un espacio centralizado.

  • Comentarios en tarjetas

    Agrega comentarios y discusiones directamente en cada tarjeta para mantener el contexto relevante.

  • Actualizaciones en tiempo real

    Los cambios se sincronizan instantaneamente mediante WebSocket para que todos vean la informacion mas reciente.

  • Permisos y acceso

    Controla quien puede ver y editar cada tablero con un sistema de permisos flexible y seguro.

Ediciones

Elige la edicion que se adapte a tus necesidades

Focalboard esta disponible en dos ediciones gratuitas para uso personal y desarrollo.

Personal Desktop

Aplicacion de escritorio para un solo usuario. Ideal para tareas personales y proyectos individuales.

  • Disponible para macOS, Windows y Linux
  • Base de datos SQLite integrada
  • Sin necesidad de servidor
  • Todas las vistas incluidas
  • Importacion y exportacion de datos

Personal Server

Servidor independiente multiusuario para equipos de desarrollo y uso personal avanzado.

  • Soporte multiusuario completo
  • Compatible con PostgreSQL y MySQL
  • Auto-hospedado en tu infraestructura
  • Sincronizacion en tiempo real via WebSocket
  • API REST documentada
  • Sistema de permisos y roles
Tecnologia

Construido con tecnologias modernas

Multilingue

Soporte nativo para multiples idiomas. La interfaz se adapta al idioma de cada usuario con archivos de localizacion i18n.

Auto-hospedado

Despliega en tu propia infraestructura y mantiene el control total de tus datos. Compatible con Docker, Ubuntu y mas.
